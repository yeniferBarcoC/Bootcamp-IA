*IA: (Semana 5 Sesion 18 ) pag. 182 https://view.genially.com/669adcb062665c97aefe5917/presentation-c7-inteligencia-artificialnivel-exploradorg2
	- Trabajo colarito en Github: git push - git pull
		Cambiar navegador predeterminado
		Abrir archivo con git

		------------------- si no hemos hecho comit por primera vez
		git init 	-> inicializa el repocitorio solo se hace una vez
		git branch -M main	-> Cambiamos a la rama main
		git remote add origin https://github.com/Jessicavallejoh/diabetes.git 	-> añadimos el enlace del repositorio
		git remote -v 	-> para ver a que repositorio apunta el proyecto

		------------------- Si vamos a clonar un proyecto
		git clone https://github.com/Jessicavallejoh/diabetes.git -> para clonar un repositorio

		------------------- Creamos el commit de los cambios
		git add yenifer.py 	-> añade un archivo especifico al repositorio
		git commit -m "Abordamos diccionarios"		-> Creamos un comit de los cambios hechos

		-------------------Subimos los cambios al repositorio en github
		git push -u origin main 	-> empujamos los datos al repositorio
		git pull origin main --allow-unrelated-histories 	->baja forzadamente las actualizaciones
		git pull origin main --rebase 	->descarga los cambios cambios del repositorio, y implementa primero los cambios antes que el de nosotros para no crear conflictos
